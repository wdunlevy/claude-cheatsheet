CLI — claude [options] [prompt]

-p, --print Non-interactive output (pipes)
-c, --continue Continue last conversation
-r, --resume [id] Resume by session ID
-w, --worktree [name] Isolated git worktree
-n, --name <name> Set session display name
--model <model> opus / sonnet / haiku
--effort <level> low / medium / high
--agent <agent> Use specific agent
--chrome Enable Chrome integration
--add-dir <dirs> Additional directories
--system-prompt <text> Override system prompt
--append-system-prompt Append to system prompt
--settings <file|json> Load settings
--mcp-config <files> MCP server configs
--plugin-dir <path> Load plugins from dir
--tools <tools> Restrict available tools
--permission-mode <m> default / plan / auto / …
--dangerously-skip-permissions Bypass checks
--bare Minimal mode (no hooks/LSP)
--output-format <fmt> text / json / stream-json
--json-schema <schema> Structured output
--max-budget-usd <$> Spending cap
--from-pr [pr] Resume PR-linked session
--fork-session New ID when resuming
--tmux tmux session (with -w)
-d, --debug Debug mode
-v, --version Show version
claude auth login / status / logout
claude doctor Health check
claude mcp add / remove / list servers
claude plugin install / enable / disable
claude install Install native build
claude update Check & install updates
